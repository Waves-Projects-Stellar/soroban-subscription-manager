### soroban-subscription-manager

A flexible smart contract system that enables developers to implement subscription-based billing models (weekly, monthly, yearly) in decentralized applications.

---

### ✨ Features

* Recurring payment scheduling
* Subscription tiers (basic, premium, etc.)
* Auto-renewal logic
* Grace periods & cancellations
* Merchant dashboard

---

### 🧱 Module Breakdown

* **Core Contracts**

  * Subscription logic
  * Payment cycles

* **Billing Engine**

  * Renewal scheduler
  * Payment validation

* **Frontend**

  * Subscription UI
  * Plan selection

* **Backend Scripts**

  * Renewal triggers
  * Payment monitoring

---

### 🏗 Technical Architecture

* Contracts: Rust + Soroban SDK
* Frontend: Next.js
* Backend: Node.js cron jobs
